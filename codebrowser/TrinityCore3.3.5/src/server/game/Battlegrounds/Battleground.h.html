549 /**
550 * @brief Pre-update hook.
551 *
552 * Will be called before battleground update is started. Depending on
553 * the result of this call actual update body may be skipped.
554 *
555 * @param diff a time difference between two worldserver update loops in
556 * milliseconds.
557 *
558 * @return @c true if update must be performed, @c false otherwise.
559 *
