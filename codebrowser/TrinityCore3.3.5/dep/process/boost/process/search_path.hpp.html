27/**
28 * Searches for an executable in path.
29 *
30 * filename must be a basename including the file extension.
31 * It must not include any directory separators (like a slash).
32 * On Windows the file extension may be omitted. The function
33 * will then try the various file extensions for executables on
34 * Windows to find filename.
35 *
36 * path must be a set of directories. Directories must be
37 * separated by colons on POSIX and by semicolons on Windows.
38 * If path is empty, the environment variable PATH is used.
39 *
40 * \returns the absolute path to the executable filename or an
41 * empty string if filename isn't found
42 *
43 * \throws std::runtime_error if path is empty and no environment
44 * variable PATH exists
45 */
46string_type search_path(const string_type &filename, string_type path = "");
