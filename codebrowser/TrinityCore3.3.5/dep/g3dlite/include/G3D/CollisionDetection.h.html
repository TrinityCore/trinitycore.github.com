37/**
38 Collision detection primitives and tools for building
39 higher order collision detection schemes.
40
41 These routines provide <I>moving</I> and static collision detection.
42 Moving collision detection allows the calculation of collisions that
43 occur during a period of time -- as opposed to the intersection of
44 two static bodies.
45
46 Moving collision detection routines detect collisions between
47 <I>only</I> static primitives and moving spheres or points. Since the
48 reference frame can be user defined, these functions can be used to
49 detect the collision between two moving bodies by subtracting
50 the velocity vector of one object from the velocity vector of the
51 sphere or point the detection is to occur with. This unified
52 velocity vector will act as if both objects are moving simultaneously.
53
54 Collisions are detected for single-sided objects only. That is,
55 no collision is detected when <I>leaving</I> a primitive or passing
56 through a plane or triangle opposite the normal... except for the
57 point-sphere calculation or when otherwise noted.
58
59 For a sphere, the collision location returned is the point in world
60 space where the surface of the sphere and the fixed object meet.
61 It is <B>not</B> the position of the center of the sphere at
62 the time of the collision.
63
64 The collision normal returned is the surface normal to the fixed
65 object at the collision location.
66

367 /**
368 Calculates the depth of penetration between a fixed sphere and a fixed
369 box as well as the deepest point of the sphere that penetrates the box
370 and the normal at that intersection.
371
372 @note There are three possible intersections between a sphere and box.
373 - Sphere completely contained in the box
374 - Sphere intersects one edge
375 - Sphere intersects one vertex
376
377 The contact point and contact normal vary for each of these situations.
378 - Sphere contained in Box:
379 - Normal is based on side of least penetration (as is the depth calculation).
380 - Point is based on center of sphere
381 - Sphere intersects one edge
382 - Normal is based on vector from the box center to the point of depth.
383 - Point is closest point to the sphere on the line
384 - Sphere intersects one vertex
385 - Normal is based on vector from the box center to the vertex of penetration.
386 - Point is vertex of penetration.
387
388 @cite Adapted from Jim Arvo's method in Graphics Gems
389 See also http://www.win.tue.nl/~gino/solid/gdc2001depth.pdf
390
391 @param sphere Fixed Sphere.
392 @param box Fixed Box.
393 @param contactPoints Sphere point that penetrates the box. [Post Condition]
394 @param contactNormals Normal at the penetration point. [Post Condition]
395
396 @return Depth of penetration. If there is no intersection between the
397 objects then the depth will be a negative value.
398 */
399 static float penetrationDepthForFixedSphereFixedBox(
400 const Sphere& sphere,
401 const Box& box,
402 Array<Vector3>& contactPoints,
403 Array<Vector3>& contactNormals = ignoreArray);

496 /**
497 Calculates time between the intersection of a moving point and a fixed
498 triangle.
499
500 @note This is only a one sided collision test. The side defined by
501 the triangle's surface normal is the only one tested. For a two sided
502 collision, call the function once for each side's surface normal.
503
504 @param orig Moving point.
505 @param dir Point's velocity.
506 @param v0 Triangle vertex 1.
507 @param v1 Triangle vertex 2.
508 @param v2 Triangle vertex 3
509 @param location Location of collision. [Post Condition]
510 (Infinite vector on no collision)
511
512 @return Time til collision. If there is no collision then the return
513 value will be inf().
514 */

516 const Vector3& orig,
517 const Vector3& dir,

519 const Vector3& v1,
520 const Vector3& v2,
521 Vector3& location) {

525 }
526 return t;

529 /**
530 Calculates time between the intersection of a moving point and a fixed
531 triangle.
532
533 @note This is only a one sided collision test. The side defined by
534 the triangle's surface normal is the only one tested. For a two sided
535 collision, call the function once for each side's surface normal.
536
537 @param orig Moving point.
538 @param dir Point's velocity.
539 @param tri Fixed triangle.
540 @param location Location of collision. [Post Condition]
541 (Infinite vector on no collision)
542 @param normal Triangle's surface normal. [Post Condition]
543
544 @return Time til collision. If there is no collision then the return
545 value will be inf().
546 */
547 inline static float collisionTimeForMovingPointFixedTriangle(
548 const Vector3& orig,
549 const Vector3& dir,
550 const Triangle& tri,
551 Vector3& location = ignore,
552 Vector3& normal = ignore) {
553
554 float t = collisionTimeForMovingPointFixedTriangle(
555 orig, dir, tri.vertex(0), tri.vertex(1), tri.vertex(2));
556
557 if ((t < finf()) && (&location != &ignore)) {
558 location = orig + dir * t;
559 normal = tri.normal();
560 }
561 return t;
562 }

565 Calculates time between the intersection of a moving point and a fixed
566 triangle.
567
568 @note This is only a one sided collision test. The side defined by
569 the triangle's surface normal is the only one tested. For a two sided
570 collision, call the function once for each side's surface normal.
571
572 @param orig Moving point.
573 @param dir Point's velocity.
574 @param v0 Triangle vertex 1.
575 @param v1 Triangle vertex 2.
576 @param v2 Triangle vertex 3
577 @param location Location of collision. [Post Condition]
578 (Infinite vector on no collision)
579 @param normal Triangle's surface normal. [Post Condition]
580
581 @return Time til collision. If there is no collision then the return
582 value will be inf().
583 */
584 inline static float collisionTimeForMovingPointFixedTriangle(
585 const Vector3& orig,
586 const Vector3& dir,
587 const Vector3& v0,
588 const Vector3& v1,
589 const Vector3& v2,
590 Vector3& location,

596 }
597 return t;
598 }
