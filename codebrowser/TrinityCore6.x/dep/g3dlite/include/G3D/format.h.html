23/**
24 Produces a string from arguments of the style of printf. This avoids
25 problems with buffer overflows when using sprintf and makes it easy
26 to use the result functionally. This function is fast when the resulting
27 string is under 160 characters (not including terminator) and slower
28 when the string is longer.
29 */
30std::string __cdecl format(
31 const char* fmt
32 ...) G3D_CHECK_PRINTF_ARGS;
33
34/**
35 Like format, but can be called with the argument list from a ... function.
36 */
