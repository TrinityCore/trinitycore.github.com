34/**
35This is the main Class to manage loading and unloading of maps, line of sight, height calculation and so on.
36For each map or map tile to load it reads a directory file that contains the ModelContainer files used by this map or map tile.
37Each global map or instance has its own dynamic BSP-Tree.
38The loaded ModelContainers are included in one of these BSP-Trees.
39Additionally a table to match map ids and map names is used.
40*/
